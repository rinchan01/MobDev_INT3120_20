# MobDev_INT3120_20
A game application that applies weekly lectures of Mobile Development course at University of Engineering and Technology, Vietnam National University (2023 - 2024).
## Introduction 
This is a monorepo for the first application assignment in Mobile Application Development course (2324II_INT3120_20) developed by group 6 with four members. 
## Compatibility 
The minimum API level required to run the app: 24. 
The API level used to test the app: 34. 
Best screen display size: 393dp x 808dp. 
## Features
1. Log in the game with unique username.
2. Play infinite game with scores counted.
3. Adjust settings for personal preferences.
4. Check leaderboard.
5. Change skin before playing game.
